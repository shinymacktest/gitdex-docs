# Styling and Theming

This section details the styling and theming implementation in SafeDocs, focusing on the use of CSS, themes (light and dark), and overall visual presentation. The project utilizes Tailwind CSS alongside custom CSS to achieve a consistent and appealing user interface.

This CSS file first imports Tailwind CSS and `tw-animate-css`, providing utility classes and animations, respectively. Then, it defines CSS variables using the `@theme inline` block, allowing easy customization of colors, fonts, and spacing.  Crucially, it defines both the default `:root` variables and overrides within the `.dark` class, enabling theme switching based on the presence of the `dark` class on the `<html>` element. The `@layer base` block applies base styles to all elements, setting default border colors and background/foreground colors based on the defined CSS variables.
