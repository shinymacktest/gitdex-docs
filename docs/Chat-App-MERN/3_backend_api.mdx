# Backend API Documentation

The backend exposes several RESTful APIs for various functionalities:

## Authentication

* `/api/auth/signup`: Creates a new user account.
* `/api/auth/login`: Logs in an existing user.
* `/api/auth/logout`: Logs out the current user.
* `/api/auth/google`: Initiates Google OAuth flow.
* `/api/auth/google/callback`: Google OAuth callback endpoint.
* `/api/auth/checkAuth`: Checks authentication status and returns user details.
* `/api/auth/checkUsernameAvailability`: Checks the availability of a username.
* `/api/auth/updateProfile`: Update User Profile


## Friends Management

* `/api/friends/sendFriendRequest`: Sends a friend request.
* `/api/friends/acceptFriendRequest/:senderId`: Accepts a friend request.
* `/api/friends/rejectFriendRequest/:senderId`: Rejects a friend request.
* `/api/friends/removeFriend/:friendId`: Removes a friend.
* `/api/friends/getFriends`: Retrieves the current user's friends.
* `/api/friends/getPendingRequests`: Retrieves pending friend requests for the user.
* `/api/friends/getSentRequests`: Retrieves sent friend requests from the user.

## Messaging

* `/api/messages/getUsersForSidebar`: Gets users for sidebar display, excluding the logged-in user.
* `/api/messages/getMessages/:id`: Retrieves chat messages between the current user and a specified user.
* `/api/messages/sendMessage/:id`: Sends a message to a specified user.
