# Frontend Components

<TOC />

This section details the React components that make up the user interface of the chat application.

## Chat Container (`ChatContainer.jsx`)

The `ChatContainer` component is the main container for the chat interface. It fetches and displays messages, handles scrolling to the bottom, and incorporates other components.

```jsx
// Key snippet from ChatContainer.jsx
useEffect(() => {
    getMessages(selectedUser._id);
    subscribeToMessages();

    return () => unsubscribeFromMessages();
}, [selectedUser._id, getMessages, subscribeToMessages, unsubscribeFromMessages]);
```

[Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/ChatContainer.jsx)


This component uses React hooks (`useEffect`) to manage fetching and subscribing to messages, ensuring that the chat is updated in real-time.

## Other Key Components

*   **`ChatHeader.jsx`**: Displays the selected user's information and a close button.  [Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/ChatHeader.jsx)
*   **`FriendsBox.jsx`**:  Manages friend requests and displays the user's friend list. [Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/FriendsBox.jsx)
*   **`MessageInput.jsx`**: Allows users to send text messages and images. [Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/MessageInput.jsx)
*   **`Navbar.jsx`**: Provides navigation and user management features. [Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/Navbar.jsx)
