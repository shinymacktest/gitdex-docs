## Introduction
This section provides an overview of the foundational setup for the Chatty application's React frontend. It covers the project's entry point, global styling, API client configuration, utility functions, and shared constants. Understanding these core elements is crucial for comprehending how the application is structured and interacts with the backend.

## Project Entry Point
The application starts execution from `main.jsx`, which is responsible for rendering the root React component, `App.jsx`, into the DOM. It also wraps the application with `BrowserRouter` for client-side routing and `StrictMode` for development-time checks.

The global styles are imported from `index.css`, which sets up Tailwind CSS and imports a custom font, `Chivo`.

```jsx
// frontend/src/main.jsx
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import './index.css'
import App from './App.jsx'
import { BrowserRouter } from 'react-router-dom'

createRoot(document.getElementById('root')).render(
  <StrictMode>
    <BrowserRouter>
      <App />
    </BrowserRouter>
  </StrictMode>,
)
```
[Source: `main.jsx`](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/main.jsx)

```css
/* frontend/src/index.css */
@import url('https://fonts.googleapis.googleapis.com/css2?family=Chivo:ital,wght@0,100..900;1,100..900&display=swap');
```

## API Client Configuration
The `axiosInstance` is a pre-configured Axios client used for making HTTP requests to the backend API. It dynamically sets the `baseURL` based on the environment (development or production) and ensures `withCredentials` is enabled for handling cookies (e.g., for authentication).

```javascript
// frontend/src/lib/axios.js
import axios from "axios";

export const axiosInstance = axios.create({
    baseURL: import.meta.env.MODE == "development" ? "http://localhost:5001/api": "/api",
    withCredentials: true,
});
```
[Source: `axios.js`](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/lib/axios.js)

## Utility Functions
The `utils.js` file contains helper functions that can be reused across different components. Currently, it provides `formatMessageTime`, which formats a given date into a human-readable time string, including year, month, day, and 12-hour time.

```javascript
// frontend/src/lib/utils.js
export function formatMessageTime(date) {
    return new Date(date).toLocaleTimeString("en-US", {
        year: "numeric",
        month: "short",
        day:"2-digit",
        hour: "2-digit",
        minute: "2-digit",
        hour12: true,
    });
}
```
[Source: `utils.js`](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/lib/utils.js)
