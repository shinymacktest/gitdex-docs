# Frontend Styling and Build Configuration

<TOC />

This section covers the frontend styling using Tailwind CSS and DaisyUI, and the build process using Vite.

## Tailwind CSS and DaisyUI

The application uses Tailwind CSS for styling, enhanced by the DaisyUI component library.  Tailwind's configuration (`tailwind.config.js`) includes numerous pre-defined themes.

```javascript
// Snippet from tailwind.config.js
daisyui : {
 themes: [
      "light",
      "dark",
      // ... many more themes
    ],
  }
```

[Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/tailwind.config.js)

## Vite Configuration

Vite is used for building and serving the React application.  The configuration (`vite.config.js`) is relatively straightforward, primarily using the `@vitejs/plugin-react` plugin.
