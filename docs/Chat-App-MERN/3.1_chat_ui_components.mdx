## Introduction
This section explores the crucial UI components that form the core of the Chatty application's user interface. These components are responsible for displaying user lists, handling chat selection, and providing visual feedback during data loading states.

## Sidebar
The `Sidebar.jsx` component is responsible for displaying a list of friends or contacts, allowing users to select a conversation partner. It integrates with `useChatStore` to fetch user data and `useAuthStore` to determine online status.

### Functionality
*   Fetches available users (friends) using `getFriends` from `useChatStore`.
*   Displays a list of users, indicating their online/offline status.
*   Allows filtering to show only online users via a checkbox.
*   Highlights the currently selected user.
*   Renders a `SidebarSkeleton` during data loading.

```jsx
// frontend/src/components/Sidebar.jsx
import { useEffect, useState } from "react";
import { useChatStore } from "../store/useChatStore";
import SidebarSkeleton from "./skeletons/SidebarSkeleton";
import { Users } from "lucide-react";
import { useAuthStore } from "../store/useAuthStore";

const Sidebar = () => {
    const { getFriends, users, selectedUser, setSelectedUser, isUsersLoading } =
        useChatStore();

    const { onlineUsers } = useAuthStore();
    const [showOnlineOnly, setShowOnlineOnly] = useState(false);
    useEffect(() => {
        getFriends();
    }, [getFriends]);
    const filteredUsers = showOnlineOnly
        ? users.filter((user) => onlineUsers.includes(user._id))
        : users;
    if (isUsersLoading) return <SidebarSkeleton />;
    // ... JSX for rendering sidebar
};

export default Sidebar;
```

## No Chat Selected
The `NoChatSelected.jsx` component serves as a friendly placeholder displayed when no specific chat conversation has been chosen by the user from the sidebar. It provides instructions to guide the user on how to start chatting.

```jsx

const NoChatSelected = () => {
  return (
    <div className="w-full sm:flex hidden flex-1 flex-col items-center justify-center p-16 bg-base-100/50">
      <div className="max-w-md text-center space-y-6">
        {/* Icon Display */}
        <div className="flex justify-center gap-4 mb-4">

  );
};

export default NoChatSelected;
```

## Skeleton Loaders
To improve user experience during data fetching, the application uses skeleton components that mimic the structure of the content while it's loading.

### Message Skeleton
`MessageSkeleton.jsx` provides a loading animation for message bubbles, indicating that messages are being fetched. It renders an array of placeholder message bubbles with avatar, header, and content skeletons.

```jsx
// frontend/src/components/skeletons/MessageSkeleton.jsx
const MessageSkeleton = () => {
    const skeletonMessages = Array(6).fill(null);
    return (
      <div className="flex-1 overflow-y-auto p-4 space-y-4">
        {skeletonMessages.map((_, idx) => (
          <div key={idx} className={`chat ${idx % 2 === 0 ? "chat-start" : "chat-end"}`}>
            <div className="chat-image avatar"> {/* Avatar skeleton */} </div>
            <div className="chat-header mb-1"> {/* Header skeleton */} </div>
            <div className="chat-bubble bg-transparent p-0"> {/* Bubble skeleton */} </div>
          </div>
        ))}
      </div>
    );
};
export default MessageSkeleton;
```
[Source: `MessageSkeleton.jsx`](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/skeletons/MessageSkeleton.jsx)

### Sidebar Skeleton
`SidebarSkeleton.jsx` is displayed when the list of contacts in the `Sidebar` is being loaded. It provides a visual structure similar to the actual sidebar content, complete with avatar and user info placeholders.

```jsx
// frontend/src/components/skeletons/SidebarSkeleton.jsx
import { Users } from "lucide-react";

const SidebarSkeleton = () => {
  const skeletonContacts = Array(8).fill(null);
  return (
    <aside className="...">
      <div className="border-b border-base-300 w-full p-5"> {/* Header */} </div>
      <div className="overflow-y-auto w-full py-3">
        {skeletonContacts.map((_, idx) => (
          <div key={idx} className="w-full p-3 flex items-center gap-3">
            <div className="relative mx-auto lg:mx-0"> {/* Avatar skeleton */} </div>
            <div className="hidden lg:block text-left min-w-0 flex-1"> {/* User info skeleton */} </div>
          </div>
        ))}
      </div>
    </aside>
  );
};
export default SidebarSkeleton;
```
