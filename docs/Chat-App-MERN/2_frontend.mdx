# Frontend Architecture

<TOC />

The frontend of the chat application is built using React.js, leveraging functional components and hooks for state management.  The UI is styled with Tailwind CSS, and Lucide React provides the icons.

## Core Components

The frontend is organized into several key components:

### 2.1 User Interface Components

*   **`Sidebar`**: Displays the list of friends, allowing users to select a chat partner.  Includes an online status indicator and a filter to show online users only.  [Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/Sidebar.jsx)
*   **`ChatContainer`**:  Displays the chat messages and the input area for sending messages.  (Implementation details omitted for brevity).
*   **`NoChatSelected`**: A component shown when no chat is selected, welcoming the user. [Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/NoChatSelected.jsx)
*   **`FriendsBox`**: (Assuming a component for friend management; not directly present in provided files).

### 2.2 Skeleton Components

Skeleton components provide loading indicators during data fetching:

*   **`SidebarSkeleton`**:  A placeholder for the sidebar while fetching friend data. [Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/skeletons/SidebarSkeleton.jsx)
*   **`MessageSkeleton`**: A placeholder for chat messages while loading.  [Source](https://github.com/shinymack/Chat-App-MERN/blob/main/frontend/src/components/skeletons/MessageSkeleton.jsx)
