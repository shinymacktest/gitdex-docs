# Frontend Implementation

<TOC />

## Project Structure

The frontend uses a component-based architecture built with React.  Key components include:

* `Navbar`: The main navigation bar.
* `HomePage`: Displays the chat interface.
* `LoginPage`, `SignUpPage`: Authentication pages.
* `SettingsPage`, `ProfilePage`: User profile and settings.

## State Management

The application leverages Zustand for state management.  This simplifies handling of authentication state (`useAuthStore`) and theme (`useThemeStore`).


## Routing

React Router is used for client-side routing, enabling navigation between different pages (home, login, signup, settings, profile).


## Styling

The project uses Tailwind CSS for styling, ensuring a consistent and responsive user interface.  DaisyUI provides pre-built components and styling enhancements.
